  -->캐릭터 상품으로 활용 가능

  활용 방안)
  -->고인과의 생전 대화 기록을 통해 현재에도 대화하고 있는 듯한 기분을 느낄 수 있음
  -->중요한 일에 대한 시뮬레이션(ex. 고백, 상견례)을 미리 돌려볼 수 있음
  입니다.

  (촬영해서 등록하면 내 취향에 맞는 한 벌 추천)

  - 옷을 클릭하면 구매할 수 있는 사이트로 연결
  - 커스텀 기능 ( 내가 등록한 옷으로 꾸미기)
  - 좋아요, 싫어요 표시 기능(싫어요 누르면 그 옷은 더이상 추천 안 함)
  입니다.
